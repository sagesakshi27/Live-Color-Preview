# 🎨 Live Color Preview

A simple Live Color Preview Snippet built using **HTML+CSS**.

> This project allows users to preview colors in real-time as they select or input values, making forms and design tools more interactive and user-friendly.

---

## 📁 Project Structure

    Live-Color-Preview/
    │
    ├── index.html        # HTML file
    ├── styles.css        # Styling
    ├── LICENSE
    └── README.md         # Documentation

---

## 💡 Features

-   Real-time background color change.
-   Color input box centered on the screen.
-   Lightweight
-   Responsive and works on all major browsers.

---

## 🧠 How It Works

1.  The user selects a color using the color picker.
2.  The background color of the webpage changes instantly using the
    `oninput` event in JavaScript.

``` html
<input type="color" oninput="preview.style.backgroundColor = this.value">
```
